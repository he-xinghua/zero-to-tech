# 认知训练小工具

一个轻量级的前端练习页面，包含多种认知训练小游戏和交互功能。

## 功能

- **深色模式** — 一键切换亮/暗主题，自动保存偏好
- **实时时钟** — 显示当前时间，每秒刷新
- **智能问候** — 根据当前时段显示不同的问候语
- **消息切换** — 点击按钮轮换多条文案
- **反应速度测试** — 等待绿色出现后快速点击，记录反应时间
- **数字广度** — 数字序列记忆力测试，逐轮递增，挑战你的工作记忆上限
- **黑猩猩测试** — 记住 1-9 在网格中的位置，按序点击，测试空间记忆
- **划消任务** — 30 秒内在字母矩阵中找到所有目标字母，训练专注力

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- CSS 自定义属性实现主题切换
- `localStorage` 持久化用户偏好和最佳成绩
